Boot Camp Management System (BCMS)

The Boot Camp Management System is designed to totally manage every aspect of the boot camp. This management includes:

  • Creating and scheduling cohorts
  • Registering students
  • Tracking student attendance
  • Assigning work

BCMS includes important features:

Cohorts

Cohorts provides flexible and comprehensive data providing all that schools need:

  • User description
  • Assignment of primary instructor
  • First and last day of the boot camp
  • Date of Demo Day
  • Maximum number of students
  • Easy enrollment and drops of students

Security

BCMS is secure. Each authorized use is required to log into the system with a username and password and is assigned authorization to access only the parts of the system they need. Features not available to a user are hidden from view.

The flexible authorization system allow configuration of multiple roles with each role capable of authorizing the user as one or multiple:

  • Admin - authorization to all functions
  • Staff - staff-only functions
  • Instructor - allows being assigned as instructor to a cohort
  • Student - allows enrolling in cohors
