About BCMS

Boot Camp Management System makes it easy to manage the entire process of delivering a boot camp cohort. The system allows administration, staff, instructors and students to keep their focus on student learning instead of all the related administration.

BCMS was created by Greg Doud, Sr. Instructor at MAX Technical Training. The technology used in BCMS is the same technology taught in the .Net Boot Camp. It includes:

Front-end (User Interface Layer) Back-end (Business Layer) Data Store (Data Layer)
Angular/Typescript C# (.Net Core 3.0) SQL Server
HTML5/CSS3/JavaScript EntityFramework Core  
Bootstrap/jQuery MVC/WebApi
